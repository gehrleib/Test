I'm not sure if there’s an "active" tag value for the RegGroup, but there is an "archive" tag. The API will exclude groups that are archived, belong to multiple books, or have no associated citations.

It’s hard to define the exact scenarios for archiving, but the archive tag allows filtering without deleting the group, preserving it for historical reference. A group might be archived if the book or a specific section no longer applies or imposes requirements on RBC.
