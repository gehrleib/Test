Introduced the ability to override the issuance jurisdiction:

In cases where an issuance is too broad (e.g., marked as "Global" or affecting all regions in CUBE), users can now manually narrow down the impacted jurisdictions.

Overrides will persist across future CUBE updates unless the override is explicitly removed.

Benefit: Reducing the number of jurisdictions lowers the number of required mappings and attestations, making the mapping process more efficient and targeted.
